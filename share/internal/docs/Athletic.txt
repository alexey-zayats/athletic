Athletic
is a sports Event Management software

Архитектура

1. Десктопное приложение под все платформы
2. Rest API, как сервер приложений
3. Сайт на AngularJS 2
4. Приложение для планшета

Зависимости
1. Qt (основная библиотека)
	1.1. qca (криптография)
	1.2. QtAv (видео/аудио обработка)
	1.3. grantlee - система шаблонов (http://www.grantlee.org/apidox/index.html)

Отчеты
1. Генератор
- получение данных
- визуализация их
- шаблоны
2. Дизайнер
- Редактор получения данных
- Редактор отображения
